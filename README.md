# Orizu Demo

**Orizu Demo** is a full-stack prototype and concept for a real-time chat application.  
It is designed to explore modern web development technologies and serve as a learning project.

## Purpose

The goal of Orizu Demo is to create a simple, real-time chat platform where users can communicate instantly.  
The project focuses on experimenting with full-stack development and modern tools, while practicing clean code and project architecture.

## Planned Tech Stack

- **Frontend:** React (Vite), TypeScript  
- **Backend:** Nest.js  
- **Database:** PostgreSQL (with Prisma ORM)  
- **Other tools:** Docker, WebSocket (Socket.io), Supabase (optional for auth)  

## Current Status

- Project is in planning phase  
- Core features and architecture are being defined  
- Development will start with MVP (real-time chat, basic UI)  

## Future Plans

- User authentication  
- Persistent chat messages  
- CI/CD and Docker-based deployment  
- Optional advanced features like notifications and analytics
